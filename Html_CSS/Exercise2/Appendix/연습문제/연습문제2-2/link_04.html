웹퍼블리싱전문가

웹 표준의 개념과 원리를 이해하고 웹 표준을 준수한 많은 사이트를 벤치마킹한 뒤 다양하고 유용한 실무 스킬과 팁을 활용한 웹 사이트(쇼핑몰,홍보사이트 등)를 제작하며, HTML+CSS+JAVA Script로 웹 컨텐츠의 구조와 표현, 동작을 명확하게 분석하고, 웹 표준을 준수하여 웹 사이트를 디자인하며, 사용자와의 상호작용을 향상시킬 수 있는 기술력을 갖춘 웹 퍼블리셔(기존의 코더라고 불리던 HTML제작자)를 양성하는 과정입니다. 구조적인 언어와 표현 언어의 관계를 인식함으로써 웹 표준 퍼블리싱을 스스로 구현하고 제작할 수 있도록 학습합니다.

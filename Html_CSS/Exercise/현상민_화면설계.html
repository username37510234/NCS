자바를 통해 계산기 프로그램을 실습하였다.
해당 프로그램은 Java 프로그램을 기반으로 만들어졌으며,
기본적인 사칙연산 외에도 '%' 버튼을 통한 나머지 계산이 가능하게 만들어졌다.
그 외에도 연속적인 계산이 가능하도록 임시 저장기능을 만들어놓았으며,
'='버튼을 누르지 않아도 계산이 되도록 기능을 추가하였다.
